  • [서버로부터 인증 응답이 없습니다.] 라는 메시지가 떠요.

    [서버로부터 인증 응답이 없습니다.] 메시지는 이용하고 계신 네트워크 환경이
    원활하지 않을 때 발생할 수 있습니다.

    아래에 안내해드리는 내용들을 점검해보신 뒤 CoDing을 다시 실행해주시길 부탁드릴게요.
    * 공유기, 핫스팟이 아닌 유선 네트워크를 이용해 게임 실행
    * 네트워크 환경 설정 확인
    * CoDing 실행 시 불필요한 프로그램 종료 후 이용

    네트워크 환경을 꼼꼼하게 점검하였는데도 같은 문제가 지속 발생한다면, 이용 중인 IP 확인 후 고객센터로 문의 부탁드립니다.

    ※ PC의 IP주소는 검색 사이트에서 [내 IP주소]를 검색해 주세요.

  • 게임 실행이 안돼요.

    게임 실행이 되지 않을 경우 게임을 삭제하신 후 다시 설치해 보시기 바랍니다.

    1. [시작버튼]-[제어판]-[프로그램 추가/제거] 에서 'CoDing'삭제
    2. [내 컴퓨터]-[C드라이브]에서 [CoDing]폴더 삭제
    3. CoDing 홈페이지 로그인 후 ‘게임시작’

    이후에도 계속 동일한 현상이 나타날 경우,
    사용하고 계신 PC사양 정보를 첨부하여 1:1문의하기로 문의해 주시기 부탁 드립니다.

    1. 오류 발생 일시 :
    2. 오류 현상 :
    3. 스크린샷 :
    4. PC 사양 정보 :

    정확한 오류 메시지, 오류 코드를 함께 적어주시면 해결에 큰 도움이 된답니다.^^.

  • 불법프로그램을 사용하는 유저를 발견했습니다 !

    불법프로그램을 사용하는 유저를 발견 시, 스크린샷을 찍어 보내주시면
    24시간 모니터링 중이기 때문에 빠른 조치가 가능합니다.
    올바른 제보라면 소소한 보상도 드리고 있으니 적극 참여 부탁드립니다.

  • 로그인 시에 "이미 접속된 유저입니다"라는 메시지가 떠요.

    "이미 접속된 유저입니다"라는 메시지는 CoDing 플레이 도중 비정상적인
    접속 종료로 인해 서버에는 접속된 상태로 인식되기 때문에 나타나는 현상입니다.

    일정시간 동안 기다리신 후 다시 한번 접속해보시고, 장시간 후에도 접속이 안될 경우
    '1:1 문의하기' 메뉴로 문의해 주시기를 바랍니다.

  • [휴면 아이디 해제] 게임ID가 휴면상태로 전환되었습니다.

    ID가 휴면상태가 되면 고객님의 개인정보를 저희 측에서는 열람하거나 활용할 수 없습니다.

    만약 서비스 이용을 다시 원하시는 경우에는 홈페이지에서 아이디/비밀번호 인증 및 휴면 해제 후
    로그인 하시면 다시 서비스를 이용하실 수 있습니다.

  • 캐시를 사용한 후 남은 금액에 대해 환불받고 싶어요.

    캐시를 충전하신 후 다음 달이 되었거나,
    일부 금액을 사용하신 경우 잔액에 대해 환불이 가능합니다.^^

    아래 내용을 확인하신 후 ‘충전하신 수단’을 기재하여 문의해주시면
    상세히 확인 후 안내를 드리겠습니다.

    ※ 환불 시 10%(1만원 이하 금액에 대해서는 1천원)의 수수료가 발생합니다.
    ※ 이벤트 참여로 지급받은 캐시는 잔액 환불로 도움을 드릴 수 없습니다.
